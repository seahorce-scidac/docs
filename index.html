Introduction

ROMSX is currently under development as a next-generation version of the Regional Ocean Modeling System (ROMS). It is designed to run on a wide range of machines, from laptops to multicore CPU and hybrid CPU-GPU systems.

Adaptive Mesh Refinement

ROMSX is built on AMReX an adaptive mesh refinement software framework, which provides the underlying software infrastructure for block structured adaptive mesh refinement operations. Full AMReX documentation can be found in the user guide and in the guided tutorials and example codes
